# Go-Basics
Go Basic project for School Assembler
# What is the main objectives of this project.
   - Learn the basics to program in Go.
   - Improve your knowledge in backend
   - Improve your knowledge about creating a API
# Getting Started
- For start you need to clone git repository then you need to open terminal in path of project and wirte go run practica.go
# Prerequisites
